# React Native Crypto Wallet
React Native App for Crypto Wallet
## Features

- Create wallet
  - testnet or livenet
- Show wallet info
  - balance in crypto & USD
  - private & public keys
  - derivation strategy, address type etc
- Send transaction
  - scan QRCode address
  - choose BTC fee
  - show calculated fee
- Generate address
  - show QRCode address
- List all addresses
  - explore address transactions
- Transactions history
  - sent & received
- Backup & Restore wallet
  - using mnemonic
  - export as JSON
